# Overview
Changes the desktop wallpaper at specified intervals, every minute. Developed in c#.

## Nuget Packages
* Costura.Fody.5.7.0
* Guna.UI2.WinForms.2.0.4.6
* HtmlAgilityPack.1.11.57
